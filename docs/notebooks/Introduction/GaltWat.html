1. The Galton-Watson Process#

One of the first applications of PGFs in biological contexts was by Galton and Watson, who was concerned about the extinction of aristrocratic family names in England, where the names were passed down from father to sons.

To investigate the probability a name goes extinct, we start by analyzing one male individual and consider sons he might have. Assume that each male has \(k\) male offspring with probability \(p_k\) and they all inherit the family name. We call the distribution of the number of male offspring the offspring distribution. We assume that the offspring distribution remains the same from generation to generation.

Definition 1.1 (The Galton Watson Process)

Consider a distribution of non-negative integers where \(p_k\) is the probability of \(k\), which we call the offspring distribution. A Galton Watson process is a sequence of random numbers \(\{X_g\}_{g \in \mathbb{N}}\) such that \(X_0=1\) and \(X_{g+1} = \sum_{j=1}^{X_g} k_{j,g}\) where each \(k_{j,g}\) is chosen independently from the offspring distribution. The numbers are indexed by generation \(g\).

In the family name context, \(X_g\) is the number of male descendants at generation \(g\), and \(k_{j,g}\) is the number of male offspring of the \(j\)’th male descendant at generation \(g\).

1.1. Applications of the Galton-Watson Process#

Although the focus on family names is perhaps frivolous, there are many other important scenarios that are mathematically equivalent. For example:

  • An invasive species has arrived at a location. Each female produces \(k\) female offspring with probability \(p_k\).

  • A virus has just entered a person’s body and begins to infect cells. Each virus produces \(k\) new viruses with probability \(p_k\).

  • An infected individual has just entered a large population. Each infected individual creates \(k\) new infections with probability \(p_k\).

  • A cell has just mutated to become cancerous. Each cancerous cell produces \(k\) new cells with probability \(p_k\) (in this case the cell either dies \(k=0\) or divides \(k=2\)).

  • A gambler buys a $1 lottery ticket. The ticket pays $k with probability \(p_k\). The gambler spends all proceeds on new tickets (with the same probabilities \(p_k\)).

Definition 1.2 (Extinction of a Galton-Watson Process)

We say a Galton-Watson Process goes extinct if \(X_g=0\) for some finite \(g\).

If the process goes extinct, it remains \(0\) for all future generations.

1.2. Extinction probability from one individual#

Let \(\alpha\) be the probability that the Galton-Watson process goes extinct. We note that each offspring at a given generation is the starting point for a new Galton-Watson process, so if the first individual has \(k\) offspring, then there are \(k\) new Galton-Watson processes that begin, each of which has probability \(\alpha\) of going extinct.

So if the initial individual produces \(k\) offspring, the probaiblity of extinction from those \(k\) offspring is \(\alpha^k\). The probability of extinction prior to knowing the value of \(k\) requires a weighted average of \(\alpha^k\). That is, the probability that there are \(k\) offspring and it goes extinct is \(p_k \alpha^k\). The extinction probability is thus the sum of \(p_k \alpha^k\).

In other words, the extinction probaiblity is \(\sum_{k=0}^\infty p_k \alpha^k = \mu(\alpha)\) where \(\mu(x) = \sum_k p_k x^k\) is the PGF of the offspring distribution.

As we’ve already stated that \(\alpha\) is the extinction probability, we must have \(\alpha= \mu(\alpha)\). This is a consistency relation. It is not guaranteed that every solution to this equation is the extinction probability, but rather that one of the solutions is the extinction probability.

Example 1.1 (Extinction Probability)

Consider the case where the number of offspring is \(1\), \(2\), \(3\), or \(4\) with equal probability. Then \(\mu(x) = (1 + x + x^2 + x^3)/4\).

Below we plot \(y=x\) and \(y=\mu(x)\) to find places that solve \(x = \mu(x)\). There are two solutions, one of which is \(x=1\) and the other \(x \approx 0.4\). The solution \(x=1\) is spurious - this is not the extinction probability. The correct solution is \(\alpha \approx 0.4\). We will learn later how to determine which is the correct solution.

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0,1,101)

def mu(x):
    return (1 + x + x**2 + x**3)/4

plt.plot(x, mu(x), color = 'blue', label = '$y=\\mu(x)$')
plt.plot(x,x, color = 'k', label = '$y=x$')
plt.xlabel('$x$')
plt.ylabel('$y$')
plt.legend()
